What is OAuth2 Contact Access?

A simple attempt to demonstrate OAuth2 usage with People API. This application is only capable of demonstrate how to access peoples contacts when user authorizes. It has been developed using Angular.

Where is the source code?

You can find the source here @ GitHub.

How does this work?

We use what known as Delegated Authorization to access user's contact list assuming user grant permissions or his or her Gmail Account grant permissions to do so as user trust google.

We send a HTTP request to Google Authorization server with few meta data. Which includes three important things. Redirect URI, Response Type, Scope. In OAuth2 Implicit Flow Grant Type Response Type should be set to Token. This means if user consents to access his or her Google account, we ask to send us an access token which we can later use to access the Resource Server. Because in order to deal with resource server we are going to need an access token.
